# Next Link

**Inovação e Tecnologia • Instituto SESI de SST • 2025**

## 📋 Descrição

Next Link é uma plataforma de comunicação corporativa desenvolvida para conectar pessoas no ambiente de trabalho por meio de conteúdos sobre inovação e tecnologia. O projeto visa inspirar, informar e engajar colaboradores através de diversos formatos de conteúdo.

## 🎯 Objetivos

- **Conexão com a Inovação**: Levar conhecimento sobre tendências em inovação e tecnologia
- **Visão Aprimorada**: Gerar engajamento e capacidade de identificar soluções
- **Estímulo à Inovação**: Incentivar proposição de soluções incrementais e disruptivas

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Design responsivo com variáveis CSS e animações
- **JavaScript ES6+**: Interatividade moderna e otimizada
- **Tailwind CSS**: Framework CSS para estilização rápida
- **Font Awesome**: Ícones vetoriais
- **Google Fonts**: Tipografia (Inter + Playfair Display)

## 📱 Responsividade

O projeto foi desenvolvido com foco em **mobile-first**:

- **Desktop**: 1200px+ (design completo)
- **Tablet**: 768px-1199px (adaptações de layout)
- **Mobile**: até 767px (otimizado para smartphones)
- **Mobile Small**: até 480px (ajustes para telas pequenas)

## ⚡ Performance

### Otimizações Implementadas:

- **Lazy Loading**: Carregamento sob demanda
- **CSS Critical**: Estilos críticos inline
- **JavaScript Modular**: Classes organizadas e otimizadas
- **Throttle/Debounce**: Otimização de eventos
- **Will-change**: Propriedades CSS para animações suaves
